Set Up Automatic Renewal
To prevent this from happening again:

# Add cron job for automatic sync
(crontab -l 2>/dev/null; echo "*/10 * * * * /usr/local/bin/sync-nginx-krb5.sh") | sudo crontab -

# Verify cron job is set
sudo crontab -l

**********************************

Monitor the Setup
You can monitor the sync with:

# Watch the sync log
tail -f /var/log/krb5-nginx-sync.log

# Check ticket status anytime
sudo -u nginx klist -c /tmp/krb5cc_nginx
